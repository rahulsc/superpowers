# Origins

Forge began as a fork of **Superpowers**, a composable skills library for AI
coding agents created by Jesse Vincent (obra). Superpowers provided a clean
foundation: modular skill definitions, hook-based activation, and a plugin
architecture for Claude Code.

On top of that foundation we built multi-agent team orchestration, verification
wiring with evidence gates, persistent project state, risk-scaled ceremony,
`.forge/` infrastructure, a pack protocol for distributing skill bundles, and a
risk classification engine. Each addition moved toward a single goal: give
engineers a structured operating mode where AI helps without surrendering
direction, memory, or reviewability.

The divergence grew large enough that tracking upstream no longer made sense.
Forge is now an independent project with its own roadmap, conventions, and
release cadence. We are grateful to Jesse Vincent for the original work that
made this possible.
